# Billeder til MJK-Judo hjemmesiden

## Placer billeder her

Følgende billeder skal tilføjes:

1. **hero.jpg** - Hero billede af træning/dojo (anbefalet: 1200x800px)
2. **born.jpg** - Børn der træner judo
3. **ungdom.jpg** - Ungdom/voksne der træner
4. **instruktor.jpg** - Billede af instruktører
5. **dojo.jpg** - Billede af lokalet/dojo'en

## Fra Instagram

Billeder kan hentes fra: https://www.instagram.com/mjkjudo/
- Download med browser extension eller værktøj som Inflact
- Gem i denne mappe
- Opdater referencer i .astro filerne

## Komprimering

Billeder bør komprimeres før upload:
- Størrelse: max 200KB per billede
- Format: JPG eller WebP
